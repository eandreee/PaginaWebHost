Modelo OSI

Capas del Modelo OSI

Capa Física

Transmisión de datos a través de medios físicos como cables o señales inalámbricas.

Capa de Enlace de Datos

Asegura la transferencia confiable de datos entre dispositivos locales.

Capa de Red

Gestiona el enrutamiento de datos entre diferentes redes usando direcciones IP.

Capa de Transporte

Proporciona transporte confiable de datos mediante protocolos como TCP y UDP.

Capa de Sesión

Establece, mantiene y finaliza sesiones de comunicación entre aplicaciones.

Capa de Presentación

Se encarga de la codificación, compresión y presentación de datos a las aplicaciones.

Capa de Aplicación

Proporciona servicios de red a las aplicaciones, como transferencia de archivos y correos.
